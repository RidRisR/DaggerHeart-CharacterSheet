# card-display-section 视觉方案详细设计

## 方案一：全屏模态式卡牌画廊 🖼️

### 视觉特点
**类似 Steam 游戏库或 Apple Photos 的网格浏览体验**

#### 触发入口设计
```
原位置显示：
┌─────────────────────────┐
│  🎴 我的卡牌 (24张)      │
│  ┌──┬──┬──┬──┬──┬──┐   │
│  │聚│职│背│领│扩│库│   │  ← 迷你徽章显示各类数量
│  │3 │5 │2 │4 │6 │4 │   │
│  └──┴──┴──┴──┴──┴──┘   │
│     [展开查看] 按钮      │
└─────────────────────────┘
```

#### 全屏模态视觉
```
┌─────────────────────────────────────────────────────┐
│ ← 我的卡牌画廊                            搜索🔍 ⚙️ │
├─────────────────────────────────────────────────────┤
│ 聚焦(3) │ 职业(5) │ 背景(2) │ 领域(4) │ 扩展(6) │ 库存(4) │
├─────────────────────────────────────────────────────┤
│                                                     │
│   ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐    │
│   │ 卡1 │  │ 卡2 │  │ 卡3 │  │ 卡4 │  │ 卡5 │    │  ← 大尺寸 image-card
│   │ 💫  │  │ ⚔️  │  │ 🏠  │  │ 🌟  │  │ 📦  │    │    丰富的视觉层次
│   └─────┘  └─────┘  └─────┘  └─────┘  └─────┘    │
│                                                     │
│   ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐    │
│   │ 卡6 │  │ 卡7 │  │ 卡8 │  │ 卡9 │  │添加 │    │  ← 包含添加按钮
│   │ 🎭  │  │ 🔮  │  │ ⚡  │  │ 🗡️  │  │ +  │    │
│   └─────┘  └─────┘  └─────┘  └─────┘  └─────┘    │
│                                                     │
└─────────────────────────────────────────────────────┘
```

#### 关键视觉优势
- **沉浸式体验**: 全屏展示，卡牌成为焦点
- **丰富的视觉层次**: 每张卡片都有足够空间展示图片和细节
- **直观的分类**: 标签页 + 数量显示，一目了然
- **专注模式**: 从主界面切换到卡牌专用空间

## 方案二：可折叠侧边画廊 📚

### 视觉特点
**类似 Figma 图层面板或 VSCode 侧边栏的可调节展示**

#### 收起状态
```
主界面 + 窄边栏：
┌─────────────────┬─┐
│                 │🎴│  ← 竖直卡牌图标
│   主要内容区     │ │
│                 │📚│  ← 可点击展开
│                 │ │
│                 │⚙️│
└─────────────────┴─┘
```

#### 展开状态（可调节宽度）
```
┌─────────────┬───────────────────┐
│             │ 🎴 卡牌集合    [×] │
│             ├───────────────────┤
│   主内容    │ 聚焦│职业│背景│领域 │  ← 紧凑标签
│             ├───────────────────┤
│   (70%)     │ ┌───┐ ┌───┐ ┌───┐ │  ← 3列网格
│             │ │卡1│ │卡2│ │卡3│ │    适中尺寸
│             │ └───┘ └───┘ └───┘ │
│             │ ┌───┐ ┌───┐ ┌───┐ │
│             │ │卡4│ │卡5│ │卡6│ │
│             │ └───┘ └───┘ └───┘ │
│   可拖拽    │                   │
│   调节  ←→  │     滚动区域      │  ← 可滚动
└─────────────┴───────────────────┘
    主区域              卡牌区域(30%)
```

#### 关键视觉优势
- **并行浏览**: 可以同时查看主界面和卡牌
- **灵活调节**: 用户可控制卡牌展示空间大小
- **上下文保持**: 不离开主界面，保持工作流连续性
- **渐进展示**: 从图标到列表到网格的层次展开

## 方案三：底部抽屉式卡牌台 🎪

### 视觉特点
**类似 iOS 地图应用的底部卡片或音乐播放器的队列展示**

#### 桌面端触发设计
在桌面端，底部抽屉需要更明确的触发方式：

**方式一：悬浮操作按钮**
```
┌─────────────────────────────────┐
│                                 │
│         主要内容区               │
│                                 │
│                                 │
│                           ┌───┐ │  ← 右下角悬浮按钮
│                           │🎴 │ │    (类似 Material Design FAB)
│                           │24 │ │    显示卡牌总数
└───────────────────────────└───┘─┘
```

**方式二：状态栏触发**
```
┌─────────────────────────────────┐
│                                 │
│         主要内容区               │
│                                 │
│                                 │
├─────────────────────────────────┤
│ 🎴 我的卡牌 (24) 聚焦3 职业5... [展开↑] │  ← 可点击的状态栏
└─────────────────────────────────┘
```

**方式三：快捷键触发**
- 键盘快捷键: `Space` 或 `C` (Cards)
- 鼠标悬停在底部边缘时显示提示
- 支持 `Esc` 快速关闭

#### 收起状态
```
┌─────────────────────────────────┐
│                                 │
│         主要内容区               │
│                                 │
│                                 │
├─────────────────────────────────┤
│ 🎴═══ 我的卡牌 (24) ═══ ↑ │  ← 可上拉的把手
└─────────────────────────────────┘
```

#### 半展开状态
```
┌─────────────────────────────────┐
│         主内容区(60%)            │
├─────────────────────────────────┤
│ 🎴 我的卡牌               ↑ ↓ │
│ 聚焦(3) 职业(5) 背景(2) 领域(4)  │  ← 横向滚动标签
├─────────────────────────────────┤
│ ┌──┐┌──┐┌──┐┌──┐┌──┐┌──┐┌──┐  │  ← 横向卡牌轮播
│ │卡││卡││卡││卡││卡││卡││+ │  │    紧凑但可见
│ │1 ││2 ││3 ││4 ││5 ││6 ││  │  │
│ └──┘└──┘└──┘└──┘└──┘└──┘└──┘  │
└─────────────────────────────────┘
```

#### 全展开状态
```
┌─────────────────────────────────┐
│      主内容区(30%)               │
├─────────────────────────────────┤
│ 🎴 我的卡牌               ↓    │
│ 聚焦(3) │ 职业(5) │ 背景(2) │... │
├─────────────────────────────────┤
│  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐│  ← 4列网格
│  │ 卡1 │ │ 卡2 │ │ 卡3 │ │ 卡4 ││    中等尺寸
│  └─────┘ └─────┘ └─────┘ └─────┘│
│  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐│
│  │ 卡5 │ │ 卡6 │ │ 卡7 │ │ 卡8 ││
│  └─────┘ └─────┘ └─────┘ └─────┘│
│                                 │
│         卡牌展示区(70%)         │
└─────────────────────────────────┘
```

#### 桌面端交互细节
1. **鼠标操作**
   - 点击悬浮按钮或状态栏打开
   - 鼠标滚轮在抽屉区域可调节高度
   - 拖拽分割线调整抽屉大小

2. **键盘操作**
   - `Space` / `C` 打开/关闭抽屉
   - `Tab` 在抽屉内导航
   - `Esc` 快速关闭

3. **视觉反馈**
   - 悬浮按钮有呼吸效果提示
   - 鼠标悬停底部边缘时显示"上拉查看卡牌"提示
   - 状态栏高亮显示当前分类数量

#### 关键视觉优势
- **渐进式展开**: 3个状态提供不同的信息密度
- **移动友好**: 天然适合触摸手势操作  
- **桌面适配**: 多种触发方式满足不同用户习惯
- **空间效率**: 不占用时可完全隐藏
- **操作直觉**: 上拉下拉的手势非常自然
- **快速访问**: 悬浮按钮提供随时可见的入口
